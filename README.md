# TODO List

API de TODO list utilizando Node.js.

## Configuração
- Copiar o arquivo .env.example para .env e informar a URL de conexão com o MySQL.
- Criar o banco de dados com o comando abaixo:
```
npm run sync
```

## Executar a aplicação
```
npm run start
```

## Corrigir coding style
```
npm run fix
```
